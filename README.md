# ToyLand

## Project Purpose
ToyLand is a modern, interactive toy marketplace application built with React and Firebase. The app allows users to browse, add, and review toys, while providing secure authentication via email/password, Google, and GitHub. It also includes a fully functional password reset feature.

## Live URL
(https://toyland-d2c08.web.app/)

## Key Features
- **User Authentication**
  - Sign up and login with email/password
  - Login with Google or GitHub accounts
  - Forgot password functionality with email reset
- **Protected Routes**
  - Add Toy
  - My Reviews
  - Only accessible to logged-in users
- **Toy Management**
  - View toys
  - Add toys (protected)
  - Submit reviews (protected)
- **Responsive UI**
  - Built with Tailwind CSS for responsive design
  - Smooth UI interactions and transitions

## NPM Packages Used
- **react** – UI library for building components
- **firebase** – Authentication and backend services
- **react-toastify** – Notifications for success/error messages
- **react-icons** – For icons such as eye/eye-slash in password fields
- **tailwindcss** – Utility-first CSS framework for styling
